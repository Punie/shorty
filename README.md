# Shorty

## Run locally

Install dependencies

```sh
yarn install
```

Migrate dev database

```sh
yarn prisma migrate dev --name init
```

Run the project

```sh
yarn dev
```

Then go to http://localhost:3000

## Technologies used

- Next.js
- Prisma
- SQLite
- React Query
- Tailwindcss
